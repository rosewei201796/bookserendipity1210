# 预置媒体文件目录

将预置频道的图片和视频文件放在这里。

## 文件命名建议

使用清晰的命名方式，例如：
- `healthy-society-video-1.mp4`
- `healthy-society-video-2.mp4`
- `healthy-society-image-1.jpg`

## 支持的格式

### 视频
- MP4 (推荐)
- WebM
- OGG

### 图片
- JPG/JPEG
- PNG
- WebP
- GIF

## 使用方法

在代码中引用时，使用路径：`/preset-media/文件名`

例如：
```typescript
imageUrl: '/preset-media/healthy-society-video-1.mp4'
```
